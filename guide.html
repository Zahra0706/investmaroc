Guide d'Utilisation de notre Site

Bienvenue sur notre plateforme ! Que vous soyez un entrepreneur ou un investisseur, ce guide vous aidera à naviguer dans notre site et à maximiser votre expérience.

Inscription : Choisissez Votre Profil

Lors de votre inscription, vous devez indiquer si vous êtes :

  • Entrepreneur : Vous avez une idée de projet mais manquez de financement. Notre plateforme vous permet de partager vos projets après validation par les administrateurs.
  • Investisseur : Vous disposez de fonds et recherchez des projets intéressants à financer. Explorez les projets disponibles et faites des demandes d'investissement.

Note : La communication entre les entrepreneurs et les investisseurs est gérée par les administrateurs pour assurer la sécurité et la transparence.

Pour les Entrepreneurs

  • Inscrivez-vous en choisissant le rôle "Entrepreneur".
  • Soumettez votre projet avec une description détaillée et le capital nécessaire.
  • Attendez la validation de votre projet par nos administrateurs.

Pour les Investisseurs

Si vous avez des fonds mais pas d'idée de projet, inscrivez-vous comme investisseur. Vous pourrez consulter différents projets classés par catégorie, avec le capital nécessaire et d'autres détails :

  • Inscrivez-vous en choisissant le rôle "Investisseur".
  • Parcourez les projets disponibles sur notre plateforme.
  • Faites une demande d'investissement pour le projet qui vous intéresse.

Processus d'Administration

Toutes les interactions entre entrepreneurs et investisseurs se font via nos administrateurs :

  • Les administrateurs valident ou rejettent les projets soumis.
  • Les demandes d'investissement sont également traitées par les administrateurs.
  • Il n'y a pas de communication directe entre entrepreneurs et investisseurs.
